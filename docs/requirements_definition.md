# 要件定義書：Denko1-Tracker（第一種電気工事士 学習トラッカー）

## 1. システムの目的
「第一種電気工事士」の資格取得に向け、自身の学習状況（筆記試験の弱点分析と技能試験のタイム測定）を視覚的に管理・把握し、限られた時間で最も効率よく合格を勝ち取るための専用学習支援Webアプリケーションを作る。

## 2. 要件定義（MVP：Minimum Viable Product）
本ポートフォリオ開発における「初期リリースで必ず動かす最小限のコア機能（MVP）」を以下のように定めます。

### 2.1 筆記試験（過去問）管理と弱点分析機能
* **固定カテゴリの導入**: 問題を「1.計算」「2.機器」「3.施工図」「4.法令」の4種類に固定する。
* **記録機能**: 何年度の過去問を解いたか、それぞれのカテゴリで何点だったかを記録できること（作成・表示）。
* **弱点特定とグラフ表示**: 各カテゴリの累計正答率を算出し、レーダーチャート（蜘蛛の巣状のグラフ）として可視化する。最も正答率の低いカテゴリを「現在の弱点」として画面に提示し、復習を促す。

### 2.2 技能試験のタイムトライアル記録機能
* **機能**: 全10問の公表問題に対して、「完成までのタイム（分）」と「合否（欠陥の有無）」の履歴を記録できること。
* **履歴確認**: 各問題ごとに過去のタイムを一覧で確認し、制限時間（60分）内に収まるようになったか、成長の軌跡を振り返れること。

### 2.3 試験カウントダウン機能
* **機能**: 学科試験日および技能試験日までの残り日数を自動計算し、ダッシュボードで「あと何日」かを可視化する。
* **表示**: ダッシュボード上部にカウントダウンパネルを配置し、学習の計画性と意欲を向上させる。
* **カスタマイズ**: ユーザーごとに「設定」画面から試験日を自由に変更・保存できること。

### 2.4 学習データの管理・エクスポート機能
* **CSV出力機能**: 蓄積した筆記試験および技能試験の全記録を CSV 形式でダウンロードできること。
* **目的**: 外部ツール（Excel等）での詳細分析や、学習データのバックアップを可能にする。

### 2.5 ユーザー認証とセキュアな個人データ管理
* **ユーザー認証**: Spring Security によるセキュアなログイン機能の実装。
* **データ分離**: ユーザーごとに個別の学習データを保持・管理し、他者のデータにアクセスできないようにする。

### 2.6 レスポンシブ対応（全画面モバイル最適化）
* **全画面対応**: ログイン、新規登録、ダッシュボード、各入力フォームのすべてにおいて、スマートフォン等の狭い画面で最適な表示（余白の削減、縦並びレイアウト）が行われること。
* **操作性の向上**: モバイル端末でのタップ操作を考慮したボタンサイズやフォーム配置の最適化。試験直前の見直しもスマホでスマートに行えるようにする。
