# 技能試験の制限時間超過時の「時間切れ」表示対応 (2026/06/25)

## 1. 課題内容
技能試験の記録において、完成タイムが制限時間（60分）を超過している場合でも、合否結果欄に「合格」または「欠陥あり」とそのまま表示されており、時間制限を守れなかったことが視覚的に判別しづらい課題があった。第一種電気工事士の技能試験では、制限時間内での完成が必須条件であるため、時間切れのレコードを際立たせる必要があった。

## 2. 実装内容
技能試験の完成タイムが60分を超えている場合、合否ステータスに関わらず結果欄に「時間切れ」と表示し、警告色（赤色）で強調表示するよう変更しました。

### 表示制御の変更
- `index.html` において、`completionTimeMinutes` が 60 を超える（`s.completionTimeMinutes > 60`）場合の条件分岐を追加しました。
- 条件に合致する場合は、合否ステータス（`isPassed`）の表示の代わりに、赤字（`class="fail"`）で **時間切れ** と表示します。
- 60分以下の場合は、従来通り `isPassed` の値に基づいて「合格」または「欠陥あり」を表示します。

```html
<span th:if="${s.completionTimeMinutes > 60}" class="fail">時間切れ</span>
<span th:unless="${s.completionTimeMinutes > 60}" th:text="${s.isPassed ? '合格' : '欠陥あり'}" th:class="${s.isPassed ? 'pass' : 'fail'}"></span>
```

## 3. 実行結果
- 技能試験の記録一覧において、完成タイムが60分以下のレコードは従来どおり「合格」または「欠陥あり」と表示されることを確認。
- 完成タイムが60分を超えるレコード（例：61分など）については、合否ステータスの登録に関わらず結果表示が「時間切れ」となり、赤字で強調されることを確認。
- これにより、本番の合格基準に則したより実践的な学習進捗の把握が可能となりました。
